# nao-driver-2026
The system includes code to be run, compiled and executed on Choregraphe 2.8.7.4, an IDE for programming NAO v6 robots. The IDE uses Python 2.7; the SDK is downloaded with the IDE. The Arduino code is to be run on an Arduino Mega 2560 Rev3. It operates a vehicle with an electrical schematic like that described in the KiCad schematic.
